🎯 DDPW Moneybook Flux - 전체 기능 기획서
📌 1. 프로젝트 개요
목적
부부 공동 가계부 웹 앱 (XLSX → Web 전환)

월별 수입/지출 관리, 카드/계좌별 집계, 예산 대비 실적 분석

기술 스택
Frontend: HTML + Vanilla JavaScript + CSS

저장소: localStorage + File System Access API (백업/복원)

개발: Cursor AI 바이브 코딩

🎯 3. 핵심 기능 로드맵
Phase 1: 기본 기능 (필수) ⭐⭐⭐
1-1. 항목관리 시스템
text
✅ 결제수단 관리
  - 신용카드, 체크카드, 계좌, 기타 구분
  - localStorage 저장/불러오기

✅ 계좌정보 관리
  - 계좌번호, 계좌명, 은행명, 비고 저장

✅ 카테고리 관리
  - 수입 카테고리 (급여, 부수입, 기타수익)
  - 지출 카테고리 (10개 대분류, 소분류별)
  - 대분류 선택 시 소분류 자동 필터링
1-2. 월별 거래 입력
text
✅ 거래 내역 입력 모달
  - 날짜 선택 (월별 범위)
  - 사용자 선택 (공용/파우/둠둠)
  - 구분 선택 (수입/지출)
  - 대분류 → 소분류 연동
  - 사용처, 세부내용, 금액
  - 결제수단 → 세부경로 연동

✅ 데이터 저장
  - localStorage에 월별 배열 저장
  - 자동 NO 순번 부여
1-3. 월별 내역 테이블
text
✅ 거래 내역 표시
  - NO, 날짜, 사용자, 구분, 대분류, 소분류, 
    사용처, 세부내용, 금액, 지출수단, 경로

✅ 기본 기능
  - 정렬 (날짜순, 금액순, 카테고리순)
  - 검색 (사용처, 세부내용)
  - 필터 (사용자, 구분, 카테고리, 결제수단)
