- **위치**: 5451, 5936, 6281, 6283, 6284, 7399, 7401, 7402 라인

```javascript
${result.summary.billingPeriod.start ? `<div>청구 기간: ${result.summary.billingPeriod.start} ~ ${result.summary.billingPeriod.end}</div>` : ''}
```

#### 후 (After)
```javascript

```
